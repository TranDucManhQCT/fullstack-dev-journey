# DEV JOURNAL

Đây là repository lưu trữ toàn bộ công pháp, tâm đắc, bút ký tu học, mảnh ghép tri thức, snippet code cùng những lần luyện công hằng ngày trên con đường hướng tới cảnh giới **FullStack Developer**.

Repository này không chỉ là nơi cất giữ code, mà còn là một phương thiên địa để ghi lại toàn bộ hành trình tu hành của bản thân. Mục tiêu là đem nơi này luyện thành một **tàng kinh các cá nhân**, có thể thấy rõ từng bước tiến cảnh của chính mình trên con đường tu đạo.

---

## 📂 Cấu trúc Repository

Để dễ quản lý và tra cứu, repository được chia thành các phần chính như sau:

- [00-journal](./00-journal/) — nhật ký học tập hằng ngày
- [01-guide](./01-guide/) — tài liệu định hướng, roadmap, ghi chú nền tảng
- [02-frontend](./02-frontend/) — HTML, CSS, JavaScript, React, Next.js
- [03-backend](./03-backend/) — Node.js, Express, API design
- [04-database](./04-database/) — SQL, PostgreSQL, Prisma
- [05-devops-cloud](./05-devops-cloud/) — Docker, Deploy, CI/CD
- [06-system-design](./06-system-design/) — kiến trúc và tư duy hệ thống
- [07-snippets](./07-snippets/) — các đoạn code ngắn có thể tái sử dụng
- [08-projects](./08-projects/) — project thực hành và project cá nhân
- [09-archive](./09-archive/) — lưu trữ tài liệu cũ
